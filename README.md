# 📡 AI Network Fault Prediction System

An end-to-end Machine Learning project that predicts network faults using telecom performance metrics.

## 🚀 Features
- Fault prediction using ML
- Streamlit dashboard for real-time input
- Confidence score output

## 🧠 Tech Stack
- Python
- Scikit-learn
- Streamlit

## 📊 Use Case
Predict faults in telecom networks to reduce downtime.
