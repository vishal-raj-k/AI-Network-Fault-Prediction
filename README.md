# 📡 AI-Based Real-Time Network Fault Prediction System

An end-to-end AI-powered monitoring dashboard that predicts network faults in real-time using machine learning and simulated streaming data.

---

## 🌐 Live Demo

👉 https://ai-network-fault-prediction-xdg4fptk8qngx7pqd4ugdm.streamlit.app/

---

## 🚀 Key Features

- 🔄 Real-time network data simulation
- 🤖 Machine Learning-based fault prediction
- 📊 Live trend visualization (CPU, Latency, Fault Probability)
- 🚨 Dynamic alert system (auto trigger & clear)
- 🎛️ Interactive Start/Stop monitoring controls
- 🧾 Live logs panel for system tracking
- ⚡ Smooth real-time updates without UI flickering

---

## 🧠 Tech Stack

- **Python**
- **Pandas, NumPy**
- **Scikit-learn (Logistic Regression)**
- **Streamlit (UI & deployment)**

---

## 📊 Problem Statement

Modern telecom networks generate massive performance data continuously.  
Traditional systems react only after failures occur.

This project focuses on:
👉 **Predicting network faults proactively** using key metrics like:
- CPU usage
- Latency
- Packet loss
- Error rate

---

## ⚙️ How It Works

1. Simulates real-time network conditions
2. Processes data using StandardScaler
3. Applies Logistic Regression model for prediction
4. Displays results through a live Streamlit dashboard
5. Triggers alerts when fault probability exceeds threshold

---

## 📈 Output

- ✅ Network Stable / ⚠️ Fault Likely
- 📊 Confidence / Probability score

- 🧾 Continuous system logs
